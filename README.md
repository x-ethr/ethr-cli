# `ethr-cli`

## Overview

`ethr-cli` provides utilities for local development, deployment(s), and ci-related management.

## Getting Started

### Installing

```bash
brew tap x-ethr/homebrew-taps

brew install ethr-cli
```

### Upgrading

```bash
brew update && brew upgrade ethr-cli
```

## Usage

###### Available Command(s)

```bash
ethr-cli --help
```

### Reinstalling

```bash
brew update
brew unlink ethr-cli
brew install ethr-cli
```
